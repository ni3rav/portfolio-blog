Stargazer was our project for DAIICT Hackout '24. It is an astronomy dashboard.

You can check out it's source code here: 
https://github.com/shravanasati/stargazer

Stargazer has several features, such as an interactive map that displays asteroid impacts on the Earth over the last year, a picture of the day, a timer for upcoming significant space events, and a chatbot powered by Gemini. The tech stack used for this project includes React in the frontend, Python for the backend, and Redis as our primary database (lmao).
