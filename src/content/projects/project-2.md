**Project Overview:**
My very first Project, a minimal css theme which has all the elements styled as per my taste. I found myself to write same style for several elements again and again so I thought to boil it down into one css file and so the eddy was created! Although it was an easy job but I've learnt a lot about css browser compatiblity and tried my best to keep eddy consistent across every browser out there. Have a look at the [Github Repo](https://github.com/ni3rav/eddy-css)
